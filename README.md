# GameCore
Unity框架整理，主要是唐老师框架以及其他地方整理，并结合实际开发进行了修改。

------

## 代码大致内容
1. EventCenter，事件中心模块
2. InputMgr，输入模块
3. JsonDataMgr，json信息模块
4. MonoMgr，mono模块
5. MusicMgr，音乐模块
6. PoolMgr，缓存池模块
7. ResMgr，Resource加载模块
8. ScenesMgr，场景加载模块
9. UIManager，UI模块(后面要被移除)
